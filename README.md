# Treehouse FSJS Techdegree: project 3 - Interactive Form
 This page contains a registration form for a hypothetical Full Stack developer conference. The user enters basic information, then decides between options for a t-shirt, picks activities and provides payment information. User input is validated both as it is entered and when the user tries to submit the form. If necessary, helpful error messages are displayed. The page reloads when all information entered is correct and the page is submitted successfully.
